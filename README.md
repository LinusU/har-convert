## Install

```sh
npm install --save har-convert
```

## Usage

```

  Usage: har-convert [options]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number

```

## API

### har-convert()

```js
var har-convert = require('har-convert')

har-convert()
```
